# Tip Jar — Frontend

React + Vite frontend for the [G$ Tip Jar](../README.md) creator tipping platform.

See the [root README](../README.md) for project overview, architecture, and full setup instructions.

## Quick Start

```bash
npm install
npm run dev
```

Make sure the [backend API](../backend/) is running on `http://localhost:5001` before using the dashboard or tipping flow.

## Scripts

| Command         | Description              |
| --------------- | ------------------------ |
| `npm run dev`   | Start development server |
| `npm run build` | Production build         |
| `npm run preview` | Preview production build |
| `npm run lint`  | Run ESLint               |
